# DeepSearch raw catalog processing


<!-- Not always needed, but a scope helps the user understand in a short sentance like below, why this repo exists -->
## Scope and usage

Example scripts which are processing some common public databases for ingestion in DeepSearch.

Processing of each example database is independent from each other, and it is organized in the [./catalogs](./catalogs)
.
